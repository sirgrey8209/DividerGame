---
description: Summarize the current session's work into a text file.
---

1. Create a `Document` folder in the current workspace root if it doesn't exist.
2. Create a new text file named `Session_Summary_[YYYY_MM_DD].txt` inside the `Document` folder.
3. In the file, write a summary of:
   - The main objective of the session.
   - The key steps taken (bullet points).
   - Any files created or modified.
   - The final status or result.
